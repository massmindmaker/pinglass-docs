Implementation Notes

  • StylesListView: Uses usePacks() hook for data fetching. 2-column grid with PackCard components featuring 2x2 collage previews.
  • StyleDetailView: Displays pack details with floating masonry collage animation (staggered delays). Fixed bottom CTA with btn-premium class.
  • TierSelectView: NEW Selected Style Block at top shows activePack from useActivePack() hook. "Изменить" button triggers onChangeStyle callback.
  • Pricing Tiers: Dynamic from pricingTiers prop. Supports discount badges with original price strikethrough.
  • Accessibility: Radio group with keyboard navigation (Arrow keys, Home/End). ARIA roles and labels on all interactive elements.
  • Badges: Featured (amber), Partner (purple), Popular (coral), Success/Discount (green).
